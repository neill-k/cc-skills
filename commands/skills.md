# CC Skills Marketplace

Welcome to the CC Skills Marketplace! This marketplace provides curated, high-quality skills to enhance your Claude Code workflows across productivity, development, testing, and documentation.

## Available Commands

### Browse All Skills
```
/skills-browse
```
View all available skills organized by category with descriptions, tags, and difficulty levels.

### Search for Skills
```
/skills-search <query>
```
Search for skills by name, description, tags, or category. Examples:
- `/skills-search testing`
- `/skills-search api design`
- `/skills-search productivity`

### Get Skill Details
```
/skills-info <skill-id>
```
View comprehensive information about a specific skill including usage guidance, related skills, and installation instructions.

### Install a Skill
```
/skills-install <skill-id>
```
Install a skill to make it available in your Claude Code environment. Once installed, mention the skill by name to use it.

## Quick Start

1. **Explore Available Skills**: Start by browsing the catalog
   ```
   /skills-browse
   ```

2. **Find What You Need**: Search for specific capabilities
   ```
   /skills-search code review
   ```

3. **Learn More**: Get detailed information about a skill
   ```
   /skills-info code-review-checklist
   ```

4. **Install & Use**: Add the skill to your environment
   ```
   /skills-install code-review-checklist
   ```

## How Skills Work

Once installed, skills are automatically available in your Claude Code environment. To use a skill:

1. **Mention it by name**: "Use the Task Breakdown skill to plan this feature"
2. **Reference it directly**: "Apply Code Review Checklist to this PR"
3. **Automatic activation**: Some skills activate automatically based on context
